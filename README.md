# SolarTrack Agent

AIoT 기반 순차제어 2축 태양추적형 미니 태양광 발전 자가진단 Agent 프로젝트다.

## 문제 정의

소형 태양광 패널은 태양 위치, 구름, 부분 음영, 오염, 과열, 충전 계통 문제에 따라 출력이 쉽게 변한다. 단순히 패널을 태양 방향으로 움직이는 것만으로는 출력 저하 원인을 알기 어렵고, 구름이나 강풍처럼 추적을 잠시 멈춰야 하는 상황도 구분해야 한다.

## 핵심 기능

- 태양 위치와 패널 방위각/고도각 시뮬레이션
- 하부 방위각 정렬 후 상부 고도각을 조정하는 순차제어
- 고정식 대비 추적식 발전량 비교
- 위치 기반 기상 정보 수집과 일사 조건 반영
- 발전량 개선률 기반 자가진단 Agent
- 오염, 음영, 구름을 고려하는 가상 비전모델 확장
- FastAPI, LangGraph, ESP32 하드웨어로 확장 가능한 구조
