# v02 태양, 센서, 발전량 모델

## 1. 태양 위치 모델

시간 범위는 6시부터 18시까지로 단순화한다. 방위각은 동쪽 -90도에서 서쪽 +90도로 이동하고, 고도각은 정오에 최대 70도가 되도록 계산한다.

## 2. 가상 조도 센서 모델

좌우 조도는 태양 방위각과 패널 방위각 차이에 따라 달라진다. 상하 조도는 태양 고도각과 패널 고도각 차이에 따라 달라진다.

## 3. 발전량 계산 모델

```text
발전량 = 최대출력 × 태양광 계수 × 각도 일치율 × 시나리오 계수 × 온도 계수 × 기상 계수
```

## 4. 온도 계수

패널 온도가 45도 이하일 때는 1.0으로 둔다. 45도를 넘으면 온도가 올라갈수록 발전량이 줄어들게 한다.

## 5. 시나리오 계수

| 시나리오 | 계수 방향 |
|---|---|
| normal | 정상 |
| cloudy | 전체 광량 감소 |
| shade | 부분 음영 |
| soiling | 오염으로 출력 감소 |
| overheat | 온도 영향 강화 |
| charging_issue | 패널 출력과 배터리 반응 불일치 |
| overload | 부하가 발전량보다 큼 |

## 6. 전압/전류 가상 계산

전압은 태양광 계수에 따라 완만하게 변하고, 전류는 `발전량 / 전압`으로 계산한다. 실제 회로 모델이 아니라 표시용 단순 모델이다.

## 7. 고정식 발전량 계산

방위각 0도, 고도각 35도를 기준으로 동일한 발전량 공식을 적용한다.

## 8. 추적식 발전량 계산

Agent가 조정한 패널 방위각과 고도각을 기준으로 발전량을 계산한다.

## 9. 기상 정보 반영

기상 계수는 구름량, 강수, 온도, 풍속을 반영한다. 강수나 강풍은 발전량 저하뿐 아니라 추적 동작 보류 판단에도 사용한다.
