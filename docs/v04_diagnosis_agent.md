# v04 진단 Agent

## 1. 역할

진단 Agent는 발전량 개선 여부와 센서, 비전, 기상 정보를 함께 보고 출력 저하 원인을 설명한다.

## 2. 센서 기반 진단 규칙

조도 센서 평균이 낮으면 광량 부족을 의심한다. 좌우 또는 상하 조도 차이가 크면 패널 방향 불일치를 의심한다.

## 3. 발전량 개선률 기반 판단

추적 후 발전량이 5% 이상 개선되면 추적 보정 성공으로 판단한다. 개선이 없으면 다른 원인을 찾는다.

## 4. 기상 정보 기반 판단

구름량이 높거나 강수 중이면 일시적인 출력 저하로 판단한다. 강풍이면 안전상 추적 제한 사유로 기록한다.

## 5. 오염, 음영, 과열, 충전 문제

| 조건 | 진단 |
|---|---|
| 조도 충분, 출력 낮음 | 오염 또는 부분 음영 |
| 패널 온도 높음 | 패널 과열 |
| 패널 출력 정상, 배터리 전압 변화 없음 | 충전 계통 문제 |
| 부하 전력이 발전량보다 큼 | 부하 과다 |

## 6. 위험도

- `normal`: 정상 또는 보정 성공
- `warning`: 점검 필요
- `danger`: 과열, 강풍 등 즉시 제한 필요

## 7. 사용자 조치 추천 문구

추천 문구는 원인을 단정하지 않고 확인 순서를 제안한다. 예: "기상 영향 가능성이 있으므로 추적을 잠시 보류하고, 날씨가 안정된 뒤 발전량을 다시 비교하세요."
