# v05 가상 비전모델

## 1. 추가 이유

발전량 저하가 각도 문제인지, 구름인지, 패널 표면 문제인지 센서값만으로 구분하기 어렵다. 비전 결과는 진단 근거를 보완한다.

## 2. 초기 구현 방식

초기에는 실제 모델을 학습하지 않는다. 시나리오에 따라 가상의 추론 결과를 반환해서 전체 구조를 먼저 검증한다.

## 3. 구름/태양 가림 검출

하늘 이미지에서 구름 또는 태양 가림 상태를 검출하는 것으로 가정한다. 구름 감지 시 추적 동작을 보류한다.

## 4. 패널 오염 검출

먼지, 물자국, 흙 등의 오염 후보를 표시한다.

## 5. 부분 음영 검출

건물, 난간, 나무 그림자 등 패널 일부를 가리는 요소를 표시한다.

## 6. 이물질/낙엽 검출

패널 표면에 놓인 이물질 후보를 표시한다.

## 7. 패널 손상 후보 검출

균열이나 표면 이상 후보를 표시하되, 초기 단계에서는 실제 손상 판정으로 단정하지 않는다.

## 8. 서보모터 동작 억제

구름이나 일시적 태양 가림이면 각도를 바꾸지 않고 대기한다. 오염이나 음영이면 추적보다 점검 안내를 우선한다.

## 9. 데이터셋 학습 방향

가상 추론 구조가 안정된 뒤 공개 데이터셋 또는 직접 촬영 이미지로 경량 모델 학습을 검토한다.
