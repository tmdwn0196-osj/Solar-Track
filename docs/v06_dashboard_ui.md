# v06 대시보드 UI

## 1. 전체 레이아웃

시뮬레이터는 한 화면에서 태양/패널 상태, 센서, 발전량, Agent 판단을 동시에 확인할 수 있게 구성한다.

## 2. 태양/패널 시각화

태양 위치, 패널 방위각, 패널 고도각을 간단한 2D 또는 CSS 기반 시각화로 표현한다.

## 3. 센서값 카드

좌우상하 조도, 전압, 전류, 발전량, 패널 온도, 배터리 전압을 표시한다.

## 4. 발전량 그래프

시간에 따른 발전량을 표시하고, 고정식과 추적식을 비교한다.

## 5. 고정식 vs 추적식 비교

현재 발전량과 누적 발전량 개선률을 함께 보여준다.

## 6. Agent 판단 카드

현재 phase, diagnosis, action, riskLevel을 표시한다.

## 7. Vision 결과 카드

구름, 오염, 음영, 이물질 후보 상태를 표시한다.

## 8. 로그 패널

기상 수집, 추적 보류, 각도 조정, 진단 결과를 시간순으로 남긴다.

## 9. 시나리오 제어 패널

사용자는 정상, 흐림, 음영, 오염, 과열, 충전 문제, 부하 과다 시나리오를 선택할 수 있다.
