# frontend

SolarTrack Agent의 React 단독 시뮬레이터다.

## 현재 포함 범위

- 태양 위치와 패널 각도 시각화
- 가상 조도 센서, 전압, 전류, 발전량 계산
- 고정식 대비 추적식 발전량 비교
- 발전량 계산 계수 표시
- 기상, Vision, Agent 상태 패널
- 진단 근거와 위험도 표시
- 위치 선택 기반 기상 수집 Agent와 수집 실패 시 시나리오 대체값 처리
- 운전 상태, Agent 단계, 위험도, 기상 계수, 누적 발전량을 한눈에 보는 대시보드 요약

## 실행

```powershell
npm install
npm run dev
```

기본 주소는 `http://127.0.0.1:5173`이다.
